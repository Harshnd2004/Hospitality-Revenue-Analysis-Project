# Hospitality Revenue Analysis Dashboard

## Short Description / Purpose
The Hospitality Revenue Analysis Dashboard is an interactive Power BI dashboard designed to analyze hotel performance across multiple cities and properties. The dashboard provides insights into revenue, occupancy, booking trends, and customer behavior, enabling hotel management to track key performance metrics and optimize revenue strategies.

This project analyzes hospitality booking data to identify revenue drivers, occupancy trends, and booking platform performance, supporting data-driven decision making in the hotel industry.

## Tech Stack
The dashboard was built using the following tools and technologies:

- Power BI Desktop – Main platform for building the interactive dashboard
- Power Query – Used for data cleaning and transformation
- DAX (Data Analysis Expressions) – Created measures for KPIs like RevPAR, ADR, and Occupancy
- Data Modeling – Relationships between fact and dimension tables for accurate analysis

## Features / Highlights

### Business Problem
Hotel chains operate across multiple cities and properties, generating large volumes of booking data. However, management often lacks a centralized view of key metrics such as revenue, occupancy, and booking platform performance. Without proper analysis, it becomes difficult to identify revenue trends, optimize pricing, and improve room utilization.

### Goal of the Dashboard
The objective of this dashboard is to:

- Track overall hotel revenue and performance metrics
- Analyze occupancy and booking trends
- Identify top revenue generating cities and properties
- Evaluate booking platform effectiveness
- Support revenue management and operational decision making

### Walkthrough of Key Visuals

#### Key Performance Indicators
The dashboard highlights important hospitality KPIs including:

- **Total Revenue:** 1.69 Billion
- **RevPAR (Revenue per Available Room):** 7,337
- **DSRN (Daily Sellable Room Nights):** 2.53K
- **Occupancy Rate:** 57.79%
- **Realisation Rate:** 70.14%
- **ADR (Average Daily Rate):** 12.70K

These KPIs help measure hotel profitability and operational efficiency.

#### Revenue by Category
A donut chart compares revenue contribution between categories such as:

- Luxury Hotels
- Business Hotels

This helps identify which segment contributes more to overall revenue.

#### Weekly Trends Analysis
A trend line chart shows RevPAR, ADR, and Occupancy % across weeks, helping management monitor performance fluctuations and seasonal patterns.

#### Booking Platform Performance
A combined chart analyzes Realisation % and ADR by booking platform, revealing which platforms generate higher value bookings.

#### Property Performance Table
A detailed table provides property-level insights including:

- Revenue
- RevPAR
- Occupancy %
- ADR
- Realisation %
- Cancellation %

This enables comparison of hotel performance across cities and properties.
## Business Impact & Insights

- Identified top-performing hotel properties and cities contributing the highest revenue.
- Revealed occupancy rate of ~57.8%, highlighting opportunities to improve room utilization.
- Helped evaluate booking platform effectiveness, enabling better distribution strategy.
- Enabled management to monitor weekly trends in RevPAR and ADR for pricing optimization.
- Provided data-driven insights to improve revenue management and operational planning.
